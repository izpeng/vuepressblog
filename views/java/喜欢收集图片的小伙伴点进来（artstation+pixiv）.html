        喜欢收集图片的小伙伴点进来（artstation+pixiv）

        Zpeng 2020-04-19 16:00:47 pixiv artstation

        本文介绍电脑vpn的配置，以及artstation和pixiv两个网站图片的批量下载方法。

        # 一、介绍

        1.artstation  (https://www.artstation.com/)

        ArtStation是综合CG视觉艺术网站，集游戏、电影、媒体和娱乐等行业的主流艺术作品展示平台。为用户展示图片、视频、模型等CG设计作品。又名A站。 这里有很多厉害的人物

        2.pixiv (https://www.pixiv.net/)

        Pixiv，是一个主要由日本艺术家所组成的虚拟社群，主体为由pixiv股份制有限公司所运营的为插画艺术特化的社交网络服务网站。新兴的日本同人画、插画作品分享站点。采用了web2.0的方式，每个参与者都有自己的主页并可以对作品评价打分。又名P站。 这里也有很多的厉害的人物啦

        # 二、翻墙

        有的小伙伴可能已经发现了，网站进不去，需要翻墙怎么办？

        1.自己搭建，购买vps（谷歌云、搬瓦工等）+安装V2Ray+客户端配置，这种方法较复杂，我也不是用的这种方法，所以这里就不多介绍了。有时间自己买个vps玩一下，再写一篇详细的教程。

        2.插件

        1）谷歌商店插件 首先安装Google Helper谷歌上网助手 可以让你访问 Google、Gmail 等，装了它你就可以自由的使用 Chrome 应用商店了。 插件地址：http://googlehelper.net/ 然后谷歌商店里搜索vpn插件

        几款vpn插件推荐：Astar VPN；1clickVPN；Free VPN；uVPN；VeeVPN。 哪个速度快，那就需要大家自己慢慢尝试了，尝试换换不同的插件，或者不同的节点。vpn插件同时只能开一个才有效果呦。 当然，有了vpn后，我们就不仅仅可以上这两个网站了，可以访问youtube、facebook等。

        # 三、图片下载

        应该有很多喜欢收集图片的小伙伴，如果我想下载A站和P站的图片，总不能一个一个点进去下载吧。下面介绍这两个网站批量下载图片的方法。

        1.artstation

        1）下载豆皮图片下载器

        2）复制artstation上某位作家的地址

        3）复制到豆皮下载器开始下载

        2.pixiv

        谷歌应用商店搜索pixiv批量下载插件 1）推荐：Powerful Pixiv Downloader

        1）插件安装后会发现p站画家页面会多出来一个图标点击。

        2）第一种方法：直接配置-抓取-下载（浏览器下载）

        3）第二种方法：复制url到第三方下载（我用的是idm） 任务-剪贴板添加-选择文件-选择位置-确定

        这样图片就会批量下载好了，怎么样，是不是很简单。 小伙伴们赶快行动起来吧！
